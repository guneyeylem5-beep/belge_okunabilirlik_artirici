# Belge Okunabilirlik Artırıcı

Bu proje, Python ve OpenCV kullanılarak belge fotoğraflarının okunabilirliğini artırmak amacıyla geliştirilmiştir.

## Özellikler
- Belge netleştirme
- Kontrast artırma
- Okunabilirlik analizi
- TXT ve PDF rapor oluşturma
- Grafiksel önce/sonra karşılaştırma
- Kullanıcı dostu arayüz

## Kullanılan Teknolojiler
- Python
- OpenCV
- Tkinter
- Matplotlib
- ReportLab

## Kullanım
```bash
python okunabilirlik.py
